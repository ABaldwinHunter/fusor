Customer Portal Proxy

Logs in and Capsules API methods from customer portal

>>>

post /fusor/api/customer_portal/login
Log in to customer portal (creates session)

Params

Param name Description
username
required

Username to customer portal account.

Validations:

  • String
password
required

Password to customer portal account.

Validations:

  • String
>>>

post /fusor/api/customer_portal/logout
Log out of customer portal (deletes session)

>>>

get /fusor/api/customer_portal/is_authenticated
Verifies an active authenticated session to the customer portal.

>>>

post /fusor/api/customer_portal/consumers
Create a new subscription consumer from RHN customer portal. See customer portal API documentation for a full list of parameters.

Params

Param name Description
name
optional

Name of the new subscription management application

Validations:

  • String
type
optional

Type of the new subscription management application (ex. satellite)

Validations:

  • String
facts
optional

Facts about the new subscription management application

Validations:

  • Hash
facts[distributor_version]
optional

Distributor version of the new subscription management application (ex. sat-6.2)

Validations:

  • String
facts[system.certificate_version]
optional

System certificate version of the new subscription management application (ex. 3.2)

Validations:

  • String
